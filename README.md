# Saurabh Shinde & Associates

Professional financial advisory website built using Angular and .NET API.

## Features
- Mutual Fund Advisory
- Retirement & SIP Planning
- Insurance & Tax Saving
- Contact form with Email API
- Fully responsive design
- Modern UI with animations

## Tech Stack
- Angular 14
- SCSS
- .NET 8 Web API
- Email integration

## Run Project
ng serve

Navigate to:
http://localhost:4200/
